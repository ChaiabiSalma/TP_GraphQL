## Description
Ce projet est une application de gestion bancaire développée avec **Spring Boot** et **GraphQL**.  
Il permet de gérer les **comptes bancaires** .

## Gestion des Comptes
- Rechercher un compte par ID.
- Créer, mettre à jour ou supprimer un compte bancaire.

## Technologies Utilisées
- **Langage** : Java 21  
- **Frameworks** : Spring Boot, Spring GraphQL  
- **Base de données** : H2 Database  
- **ORM** : Hibernate / JPA  
- **Outils GraphQL** : GraphiQL  
- **Autres outils** : Maven, IntelliJ IDEA
